可以通过CSS的伪元素 `::before` 在 `div.thinkTag` 的起始位置插入“正在思考”四个字。代码如下：

```html
<!-- HTML -->
<div class="thinkTag"></div>
```

```css
/* CSS */
.thinkTag::before {
  content: "正在思考";  /* 插入文本 */
  /* 可选样式：调整字体、颜色、间距等 */
  font-family: sans-serif;
  color: #666;
  margin-right: 8px;
}
```

### 效果说明：
1. **伪元素插入内容**：`::before` 会在 `div.thinkTag` 的内容区域**最前面**插入“正在思考”。
2. **默认布局**：文本会与 `div` 内的其他内容（如果有）水平排列，居左显示。
3. **自定义样式**：可根据需求添加 `font-size`、`color`、`margin` 等样式。

### 进阶示例（含加载动画）：
如果希望更直观，可添加一个“加载中”动画：

```css
.thinkTag::before {
  content: "正在思考";
  display: inline-flex;
  align-items: center;
  gap: 8px; /* 文字与动画的间距 */
  color: #333;
}

/* 添加加载动画 */
.thinkTag::after {
  content: "";
  width: 12px;
  height: 12px;
  border: 2px solid #ddd;
  border-top-color: #333;
  border-radius: 50%;

}

@keyframes spin {
  to { transform: rotate(360deg); }
}
```

这样会在“正在思考”右侧显示一个旋转的加载图标，增强用户体验。
